What is a script? How do i create one?

  • A script is a series of instructions that the computer can follow in order to achieve a goal.
  • Each time the script runs, it might only use a subset of all the instructions
  • Computers approach tasks in a different way that humans, so your instructions must let the computer solve the task programmatically.
  • To approach writing a script, break down your goal into a series of tasks and then work out each step needed to complete that task (a flowchart can help)

Expressions & Operators

  • An Expression evaluates into (results in) a single value. Broadly speaking there are two types of expressions. One tht Assigns a value to a variable, and one that uses two or more values to return a single value.
  • Expressions rely on things called operators. These allow programmers to create a single value from one or more value.
  • To understand Arthmetic Operators, revisit page 76 in the Duckett book on Javascript
  • To udnerstand String Operators, revist page 78 and 79 in the Duckett book on Javascript

What is a function?

Functions let you group a series of statements together to perform a specific task. If different parts of a script repeat the same task, you can reuse the function. Consult pages 88-94 in duckett for more information on functions.
